Jira Configuration

Configure your Jira connection to fetch tickets and post updates. You'll need to create an API token in your Jira account settings.

Enter your Jira domain (without .atlassian.net)
Your Jira account email address
Create an API token in your Atlassian account

How to Get Your API Token

  1. Go to Atlassian Account Settings
  2. Click "Create API token"
  3. Give your token a descriptive name (e.g., "Chrome Extension")
  4. Copy the generated token and paste it above
  5. Keep your token secure and don't share it with others

Data & Privacy

This extension stores your Jira credentials locally in Chrome's secure storage. Your data is never sent to external servers except for direct communication with your Jira instance.
